# Usual attack pipeline

## 1. Finding a server

Many minecraft voting sites contain many servers that have gambling mechanics within them, pick one of theese as a target

## 2. Plugin Recon

Alot of minecraft servers have no protection against finding the plugins that the server uses

The most common method is to download meteor client and run `.server plugins` in chat, this will output a list of plugins, example below:

![Example](./images/server_plugins.png)

You can then check [DupeDB](https://dupedb.net) to see if there are any recent exploits in the plugins that the server uses. This can be automated with the [DU Public Addon](https://github.com/TheUnsocialEngineer/du-addon-public) by running `.DupeDB plugins` if there are no recent exploits in the plugins the server uses, look for them yourself, plugins with anything to do with item storage or the economy are great places to look

## 3. After exploit discovery

After you have found the exploit, you can either give it off to someone else to use on the server to disrupt the economy, or use it yourself and mass distribute items to everyone, a good mod for this is the [Pay Everyone](https://github.com/aurickk/Pay-Everyone) mod
