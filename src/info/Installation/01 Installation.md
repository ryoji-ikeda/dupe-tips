# Installation Guide

## Modrinth Setup

[Modrinth App](https://modrinth.com) makes it easy to create separate profiles for testing servers. This lets you manage mods for multiple versions without constantly changing the mods in your main profile.



## Installation Steps

1. Go to modrinth.com (official site)  
2. Click "Get Modrinth App"

<img src="./images/gma.png" alt="Get modrint app showcase image" width="700">

3. Click Download
4. Run the installer and keep the default settings  
5. Once installation is complete, open the launcher  
6. Sign in with your Microsoft account  


## Creating Profiles

1. On the Modrinth homepage, click the plus (+) icon on the left

<img src="./images/plus.png" alt="Example" width="400">

2. Select "Custom Setup"

<img src="./images/cs.png" alt="custom setup image" width="400">

3. Choose your loader and game version
   - *Fabric is usually preferred for dupehunting*  
4. Click Create Instance
5. Wait for the profile to finish downloading  
6. Click Play once it is ready  


## Installing Mods

1. Many mods can be installed directly through Modrinth using the "Browse Content" button  
2. For external mods, click "Upload Files" and select the `.jar` file you want to install  
3. **Only install mods from trusted sources**  
   - **Never install random files sent by strangers**
